# 事件记录 (Thing Note)

一款用于追踪和记录日常事件的应用。

## 功能特点

### 事件记录
- 创建和编辑事件记录
- 设置发生时间和持续时长
- 添加文字备注
- 附加照片（拍照或相册选择）
- 录制音频笔记
- 关联事件分类标签

### 事情管理
- 自定义事情名称分类
- 快速搜索筛选分类

### 主题
- 浅色模式 / 深色模式 / 跟随系统

### 数据管理
- 本地 SQLite 数据库存储
- 一键清除所有数据
- 批量分享记录（打包为 ZIP）

## 技术栈

| 技术 | 用途 |
|------|------|
| Flutter | 跨平台应用框架 |
| Riverpod | 状态管理 |
| SQLite | 本地数据存储 |
| GoRouter | 路由导航 |
