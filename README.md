# 事件记录 (Thing Note)

一款用于追踪和记录日常事件的应用，支持文字、照片、录音和图片标注。

## 功能特点

### 事件记录

- 创建和编辑事件记录
- 设置发生时间和持续时长
- 添加文字备注
- 附加照片（拍照或相册选择）
- 附加视频（相册选择）
- 录制音频笔记 / 从文件导入音频
- 附加 GPS 位置信息（自动定位或手动输入）
- 标记提醒功能

### 图片标注

- 箭头标注
- 文本标注
- 矩形标注
- 自由绘画
- 马赛克模糊
- 可选颜色和线宽

### 事情管理

- 自定义事情名称分类
- 快速搜索筛选分类

### 多选操作

- 长按进入多选模式
- 批量分享记录（打包为 ZIP，含视频）
- 批量删除记录

### 备份管理

- 查看所有导出的备份 ZIP 文件
- 支持多选、批量删除和分享备份

### 主题与语言

- 浅色模式 / 深色模式 / 跟随系统
- 中文 / 英文 / 跟随系统

### 数据管理

- 本地 SQLite 数据库存储
- CSV 格式导出
- 一键清除所有数据

## 技术栈

| 技术 | 用途 |
|------|------|
| Flutter | 跨平台应用框架 |
| Riverpod | 状态管理 |
| SQLite | 本地数据存储 |
| GoRouter | 路由导航 |
| geolocator | GPS 定位 |
| video_player | 视频播放 |
